# Git-Task
# Git Commands 

This README provides Git Commands used in Git-Task.

# Initialize a Git Repository
```bash
git init
```

# Staging Changes
```bash
git add .
```
# Committing Changes
```bash
git commit -m "<commit_message>"
```

# Managing Branches
```bash
git branch -M main
```
# Setting Up Remote
```bash
git remote add origin git@github.com:mohamed0690/Git-Task.git
```

# Viewing Remote Information
```bash
git remote -v
```

# Pushing Changes
```bash
git push -u origin main
```

# Checking Status
```bash
git status
```
# Amending the Last Commit Message
```bash
git commit --amend -m "<new_commit_message>"
```


# Viewing Commit History
```bash
git log
```

# Cloning a Repository
```bash
git clone git@github.com:mohamed0690/Git-Task.git
```
